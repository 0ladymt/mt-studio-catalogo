MT STUDIO CATALOGO

Como publicar:
1. Suba todos estes arquivos em um repositório do GitHub.
2. Ative GitHub Pages em Settings > Pages.
3. Escolha a branch main e a pasta /root.
4. O site ficará online em um link do GitHub Pages.

Como atualizar:
- Rode novamente o gerador com uma pasta contendo MODELO XX + PNG/OBJ/YDD/YTD.
- Substitua os arquivos gerados e publique novamente.

Observação:
- A visualização 3D usa OBJ no navegador com Three.js via CDN. Para funcionar sem internet, seria necessário baixar as bibliotecas para dentro do projeto.
